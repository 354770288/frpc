# frpc一键安装脚本

安装
```shell
wget https://raw.githubusercontent.com/354770288/frpc/master/frpc_install.sh && chmod +x frpc_install.sh && ./frpc_install.sh
```

提示: 快捷命令安装后，可直接在终端输入 'frp' 启动脚本
